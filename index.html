Dharma-dynamic-css

A microframework to create dinamic css with states like hover, active and more

Download .zip Download .tar.gz View on GitHub

Welcome to Dharma Dynamic CSS page.

Sometimes we need to create style rules dynamically, what we can achieve by changing the style attribute directly in javascript. However, this method dont allow the creation of style rule with pseudo-class live hover, active, focus and more.

With this framework you can create much more styles dinamically, whenever it is needed.

How to install:

Download the script file on the links above or just declare the github cdn on the header:

<link rel="stylesheet" href="https://cdn.rawgit.com/ProjetoDharma/dharma-dynamic-css/master/dharma-dinamyc-css.js" />

How to use:

var styleBase = { width: "200px", height: "40px", "background-color": "#FFF" "box-sizing": "border-box" }

var styleHover = { "background-color": "#EEE" }

var styleLink = { "border": "1px solid #999" }

var styleVisited = { color: "rgba(255, 0, 0, 1)" }

style = { selector: "#my-element", base: styleBase, hover: styleHover, link: styleHover, visited: styleHover }

dharma.css.createCSSRule(style);



You can send the pseudo-class in the selector attribute:



var styleFirst = { width: "200px", height: "40px", "background-color": "#FFF" "box-sizing": "border-box" }

style = { selector: "p:first-child", base: styleFirst }

dharma.css.createCSSRule(style);
